# Custom Fedora Kernel (6.8.10)
Bu repo, Fedora 43 için özelleştirilmiş 6.8.10 kernel kaynaklarını ve spec dosyalarını içerir.

- **Hedef:** x86_64
- **Versiyon:** 6.8.10
- **COPR Projesi:** [https://copr.fedorainfracloud.org/coprs/hynkzz/ro-Kernel-H/]
## Kurulum
```bash
sudo dnf copr enable hynkzz/ro-Kernel-H
sudo dnf upgrade kernel
```
